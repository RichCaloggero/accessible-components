Datalist Combobox

A combobox implementation using the native <datalist> element. The browser provides keyboard control, focus management, and filtering. The component adds a live-region match count for screen readers.

How it works: Type into any input above to see the browser's native datalist filtering. A screen reader will hear a match count announced via the role="status" live region (e.g., "3 items found."). The message clears after the configured timeout (default 7 seconds).
